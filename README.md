# youtuvj2

YouTubeの動画4つをキーボード操作でリアルタイム切り替えしながら、アニメーションテキストを重ねて表示するブラウザベースVJアプリケーション

## 🎵 機能

- **YouTube動画4つの同時管理**
  - 埋め込みプレーヤーで再生
  - リアルタイム切り替え
  - 自動ミュート

- **動画切り替えシステム**
  - 自動切り替えモード（2〜10秒のランダム間隔）
  - 手動切り替え（キーボード操作）
  - トランジションエフェクト（フェード、グリッチ、スライド）

- **テキストオーバーレイ**
  - 3つのテキストレイヤー
  - 6種類のアニメーション（スクロール、縦移動、回転、点滅、ネオン、波）
  - カスタマイズ可能（色、サイズ、速度、位置）

- **キーボードコントロール**
  - 動画切り替え、エフェクト、テキスト表示制御
  - 速度調整、フルスクリーン切り替え

- **設定の保存・読み込み**
  - LocalStorageで自動保存
  - JSON形式でエクスポート/インポート

## 🚀 使い方

### 1. 起動方法

ローカルサーバーで実行してください（CORS制限のため）：

```bash
# Python 3の場合
python3 -m http.server 8000

# Node.jsの場合
npx http-server -p 8000
```

ブラウザで `http://localhost:8000` にアクセス

### 2. 初期設定

1. コントロールパネルでYouTube URLを4つ設定
2. テキストを入力（任意）
3. アニメーションと色を選択
4. 「💾 Save Config」で設定を保存

### 3. キーボードショートカット

**動画制御**
- `1-4`: 動画1〜4に直接切り替え
- `Space`: ランダム切り替え
- `Enter`: 次の動画へ順番に

**エフェクト**
- `Q`: グリッチエフェクト
- `W`: フラッシュエフェクト
- `E`: カラーシフト
- `X`: ブラックアウト
- `C`: ホワイトアウト

**テキスト**
- `A`: テキスト1 表示/非表示
- `S`: テキスト2 表示/非表示
- `D`: テキスト3 表示/非表示
- `R`: アニメーションリセット

**システム**
- `M`: オート/マニュアルモード切替
- `F`: フルスクリーン
- `H`: ヘルプ表示

**速度調整**
- `↑/↓`: 切り替え速度
- `←/→`: テキスト速度

## 📁 ファイル構成

```
/youtuvj2
  - index.html          # メインHTML
  - style.css          # グローバルスタイル
  - main.js            # アプリケーション統合
  /modules
    - VideoManager.js       # YouTube動画管理
    - TransitionEngine.js   # 切り替えエフェクト
    - TextOverlay.js        # テキストアニメーション
    - KeyboardController.js # キーボード操作
    - ControlPanel.js       # 設定UI
    - ConfigManager.js      # 設定の保存/読み込み
```

## 🎨 カスタマイズ

### 設定のエクスポート/インポート

- **エクスポート**: 「📤 Export」ボタンでJSON形式で保存
- **インポート**: 「📥 Import」ボタンでJSONファイルを読み込み

### カスタムトランジション

`TransitionEngine.js` で新しいトランジションを追加可能：

```javascript
transitionEngine.addCustomTransition('myTransition', (fromEl, toEl) => {
  // カスタムトランジション処理
});
```

## 💡 Tips

- **パフォーマンス最適化**: 全動画が自動ミュートされます
- **設定の自動保存**: LocalStorageに自動保存されます
- **フルスクリーン推奨**: `F`キーでフルスクリーンに切り替え
- **ライブ配信対応**: YouTubeライブURLも使用可能

## 🐛 トラブルシューティング

**動画が表示されない**
- YouTube URLが正しいか確認
- ブラウザのコンソールでエラーを確認
- CORS制限を回避するため、ローカルサーバーで実行

**設定が保存されない**
- ブラウザのLocalStorageが有効か確認
- プライベートモードでは保存されません

**キーボードが反応しない**
- 入力フィールド以外をクリック
- ブラウザのキーボードショートカットと競合している場合があります
